# **App Name**: ITB Swadharma Major Matcher

## Core Features:

- Interactive Psychometric Quiz: Engage users with a 10-question psychometric/interest-aptitude quiz, featuring multi-choice answers and dynamic UI transitions between questions.
- Major Matching Algorithm: Utilize a weighted scoring algorithm to process quiz answers and recommend suitable ITB Swadharma undergraduate programs by calculating a match percentage for each option.
- Dynamic Loading Experience: Display an animated 'Loading Dashboard' effect while the major matching algorithm processes results, enhancing the perception of data analysis and engagement.
- Personalized Result Dashboard: Present the primary recommended program in a prominent manner with a concise explanation, accompanied by an animated chart visualizing match percentages for all five programs.
- Printable Recommendation Card: Enable users to download an aesthetically designed result summary, including their name, program matches, and university branding, in PNG or PDF format.
- Direct Registration Call-to-Action: Provide a clear 'Daftar Sekarang' (Register Now) button on the results page, linking users to the ITB Swadharma application portal.

## Style Guidelines:

- A dark color scheme that embodies a 'tech-modern' and sophisticated campus aesthetic, providing a striking backdrop for glassmorphism effects. The overall palette is inspired by themes of technology, modernity, and guided discovery, avoiding overt corporate or academic tones.
- Primary Color: An electric and vibrant blue (#6BA9FF), symbolizing innovation, clarity, and trust. This color will be used for interactive elements, highlights, and key information, standing out boldly against the dark background.
- Background Color: A deep, subtle dark bluish-grey (#121418), offering a serene and immersive experience while enhancing the vibrancy of other elements. Its low saturation maintains a professional yet modern feel.
- Accent Color: A refreshing turquoise (#49CCE6), analogous to the primary color yet distinct in its hue. This color will be sparingly used for secondary calls-to-action, progress indicators, and visual flourishes, adding dynamic contrast and visual interest.
- Headlines and prominent text will use 'Space Grotesk' (sans-serif), chosen for its modern, slightly technical, and impactful character that aligns with the 'tech-modern' aesthetic. Body text, question options, and longer descriptions will utilize 'Inter' (sans-serif) for its high readability and clean, neutral appearance across all screen sizes.
- The design will follow a mobile-first approach, ensuring a clean, responsive layout across all devices. Elements will favor centered or single-column layouts for small screens, adapting gracefully to larger viewports. Glassmorphism will be selectively applied to cards and interactive panels for depth and visual appeal.
- Utilize Framer Motion extensively for smooth, deliberate UI animations. This includes seamless 'slide in/out' or 'fade in' transitions between quiz questions, an engaging and dynamic progress bar that animates with each answer, and a distinctive 'scanning/analyzing' animation for the 'Loading Dashboard' state, giving the impression of complex data computation. Chart elements will animate from 0% to their target values on display.
